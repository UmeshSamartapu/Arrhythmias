# 4. What “Normal Sinus Rhythm” Means

Normal sinus rhythm means:

* The **SA node (natural pacemaker)** controls the heartbeat.
* The rhythm is **regular**.
* Heart rate is **within normal range**.

This is a **healthy rhythm pattern**.

Signs in ECG:

* P wave before every QRS
* Regular rhythm
* Consistent intervals

Your ECG shows these.

✔ Good sign.

---

# 5. Understanding the Leads

An ECG has **12 leads** that look at the heart from different angles.

### Limb Leads

* I
* II
* III
* aVR
* aVL
* aVF

### Chest Leads

* V1
* V2
* V3
* V4
* V5
* V6

Each lead shows a **different wall of the heart**.

| Leads        | Heart Area |
| ------------ | ---------- |
| V1–V2        | Septal     |
| V3–V4        | Anterior   |
| V5–V6        | Lateral    |
| II, III, aVF | Inferior   |

### What is ischemia?

Ischemia means:

**Reduced blood flow to heart muscle.**

Possible causes:

* Coronary artery blockage
* Narrowed arteries
* Reduced oxygen supply

Signs in ECG include:

* ST depression
* T-wave inversion
